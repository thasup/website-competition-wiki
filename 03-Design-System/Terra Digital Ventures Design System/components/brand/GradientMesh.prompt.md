GradientMesh — the signature hero backdrop. Wrap any marketing hero; it renders the earth-to-sky wash and the focal Orb. Non-negotiable on marketing heroes.

```jsx
<GradientMesh height={520} fade="#ffffff">
  <div className="hero-content">…headline, subhead, CTA…</div>
</GradientMesh>
```

Props: `height`, `orb`/`orbSize`/`orbTop`/`orbLeft`, `fade` (set to the color of the section directly below so the wash blends). Place a max-width content container inside as children.
