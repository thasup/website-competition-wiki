Input — text field with a Cerulean focus ring. Use in forms, search bars, and sign-up flows.

```jsx
<Input placeholder="you@company.com" type="email" />
<Input prefix="$" type="number" defaultValue="0.00" />
<Input invalid placeholder="Required" />
```

Props: `size` (`md`/`sm`), `invalid` (Terra Cotta border), `disabled`, `prefix`/`suffix` adornments. Numeric inputs use tabular figures automatically.
