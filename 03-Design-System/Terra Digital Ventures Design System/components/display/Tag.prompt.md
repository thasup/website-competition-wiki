Tag — small uppercase pill for labels, categories, and status.

```jsx
<Tag>New</Tag>
<Tag tone="laurel">Live</Tag>
<Tag tone="tawny">Beta</Tag>
```

Tones: `cerulean` (default), `neutral`, `laurel`, `tawny`, `sienna`. Earth tones are reserved for data-viz legends and status — never as button substitutes.
