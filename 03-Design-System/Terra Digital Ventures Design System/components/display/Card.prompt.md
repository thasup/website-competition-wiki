Card — surface container in four treatments. Compose feature grids, pricing tiers, and product mockups from it.

```jsx
<Card variant="feature">Feature explanation…</Card>
<Card variant="featured">Inverted dark tier…</Card>
<Card variant="warm">Warm interlude…</Card>
<Card variant="mockup" interactive>Product UI…</Card>
```

Variants: `feature` (white + hairline + subtle shadow), `featured` (inverted Taupe — the distinctive featured/pricing tier), `warm` (Light-Tawny band), `mockup` (16px chrome, deeper shadow). `interactive` adds hover lift.
