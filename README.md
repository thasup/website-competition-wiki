# Terra Digital Ventures — Website Hackathon 2026

Planning & build wiki for the **TDV Website Hackathon 2026**: a 10-day competition to redesign the [Terra Digital Ventures](https://www.terradigitalventures.com) marketing website, judged equally on the business story and on how thoroughly Claude was used to build it.

This repo is an **Obsidian vault** (open it as a vault for the full experience — wikilinks, tags, graph view). It is currently the planning/research/design layer; the application codebase has not been scaffolded yet.

- **Deadline:** Wed 15 Jul 2026 23:59 ICT
- **Demo Day:** Thu 16 Jul 2026, 10:00–12:00
- **Start here:** [00-Home/00-Home.md](00-Home/00-Home.md)

## Structure

Numbered folders read top-to-bottom as the project flows: research → build → present.

| Folder | Contents |
|---|---|
| `00-Home` | Dashboard / map of everything |
| `01-Project` | Charter, competition rules, timeline, requirements tracker, roles |
| `02-Brand-Positioning` | Business story & positioning work (25% of score) |
| `03-Design-System` | Official TDV design system + brand kit |
| `04-Product` | Sitemap, IA, per-page specs, UX notes |
| `05-Build` | Tech stack decisions, repo setup runbook, deployment/submission mechanics |
| `06-Claude-Playbook` | Claude usage log + prompt library (25% of score) |
| `07-Presentation` | Demo Day plan & slide outline |
| `08-Resources` | Reference PDFs, inspiration teardowns |
| `09-Logs` | Daily standup / decision logs |
| `99-Archived` | Superseded material |
| `_templates` | Note templates |

See [CLAUDE.md](CLAUDE.md) for detailed conventions and guidance on working in this repo (including with Claude Code).
